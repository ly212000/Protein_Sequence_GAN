Cách chạy chương trình:
Chỉ cần chạy câu lệnh sau:
python demo.py
Chương trình sẽ tạo ra 2 file text: generated_sequence.txt và log_result.txt lần lượt để ghi ra chuỗi trình tự protein mà nó tạo ra, và ghi kết quả trong quá trình training của mỗi epoch.
Kết thúc quá trình GAN, trình tự được lưu trong file generated_sequence.txt. 

Các packages cần cài đặt:
python 3.7
codecs
numpy
tensorflow==1.15.0
subprocess
gzip
